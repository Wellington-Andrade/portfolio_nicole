# Notas sobre React Native, Skia, Expo MeshGradient e Reanimated

Este site foi implementado como web app em React + Motion, porque o alvo atual é um portfólio para navegador.

As tecnologias React Native citadas são melhores para um app Expo separado:

- `@shopify/react-native-skia`: gráficos 2D com Skia em React Native.
- `expo-mesh-gradient`: view nativa de mesh gradient em Expo.
- `react-native-reanimated`: animações nativas; no SDK atual da Expo a versão empacotada é 4.3.1.
- `react-native-svg`: base comum para SVG animável no React Native com Reanimated.

Um caminho futuro seria criar uma pasta `portifolio_nicole_app` com Expo e reaproveitar a direção visual deste site como versão mobile/instalação.
